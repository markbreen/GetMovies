The Problem

Why Does This Exist?

Have you ever tried to search for a movie on The Pirate Bay?

It's a nightmare. You just want to find a movie, but instead you're playing whack-a-mole with popup windows, fake download buttons, and sketchy redirects. Every click spawns three more tabs you didn't ask for.

This Script Solves That Problem

No browser. No popups. No ads. No malware.
Just a clean terminal interface that fetches what you need.

Getting Started

System Requirements

Required (1985)

  • Intel 80486 DX2 66MHz or better
  • 640KB RAM (512KB free conventional)
  • VGA display (256 colors recommended)
  • 56K modem (28.8K minimum)
  • MS-DOS 6.22 with Windows 3.11
  • A parent who doesn't pick up the phone

Actually Required

  • Windows with PowerShell 5.1+
  • Internet connection
  • A torrent client (for magnet links)
  • Appreciation for retro aesthetics
