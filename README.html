üöÄ Quick Start

Option 1: Download Binary (Recommended)

Download the pre-built binary for your platform from GitHub Releases:

# Windows
Download getmovies-windows-amd64.exe and double-click

# macOS
curl -LO https://github.com/markbreen/GetMovies/releases/download/v3.0/getmovies-macos-arm64
chmod +x getmovies-macos-arm64
./getmovies-macos-arm64

# Linux
wget https://github.com/markbreen/GetMovies/releases/download/v3.0/getmovies-linux-amd64
chmod +x getmovies-linux-amd64
./getmovies-linux-amd64

Option 2: Build from Source

# Clone repository
git clone https://github.com/markbreen/GetMovies.git
cd GetMovies

# Build
go build -o getmovies

# Run
./getmovies

üéÆ How to Use

1. Search

  • Launch the app
  • Select [1] SEARCH
  • Enter movie name
  • View results instantly

2. Get Magnet Link

  • Select result number (1-10)
  • View full magnet link
  • Copy to clipboard
  • Paste in torrent client

3. Quick Re-run

  • Select [2] RECENT SEARCHES
  • Choose from last 10
  • Auto quick mode if recent
  • Lightning fast results

‚å®Ô∏è Keyboard Shortcuts

NAVIGATION:
B or b - Go back to previous menu

SEARCH:

[20] - Show 20 results

[1-10] - View magnet link

HISTORY:
[1-10] - Re-run a recent search
C or c - Clear all history (with confirmation)
